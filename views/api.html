Welcome To our API Documentation

URL HTTP Verb POST Body Response
/api/quotes GET Empty Return all the quotes in the database.
/api/quotes POST JSON String with the fields "quote" and "author". Add a new quote to the DB.
/api/quotes/like/:id GET Empty Return a single quote whose id is the parameter id
/api/quotes/:id PUT Empty Update by one the like field of a quote whose id is provided in the parameter id.
/api/quotes/:id DELETE Empty Delete the quote whose id is provided.
/api/authors GET Empty Return the name of all the authors in the database.
/api/authors/:author_Name GET Empty Return all quotes by author author_Name
/api/quotes/trend GET Empty Return the top 10 most liked quotes.
/api/quotes/search/:term GET Empty Return all quotes that contain the search term "term".
